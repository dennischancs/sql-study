# 1. 搭建mysql环境
1. 安装docker engine（linux） 或 docker desktop（windows/macOS）
2. 下载本仓库的`docker-compose.yml`文件，同目录下 运行 `docker-compose up -d` 命令，完成启动
3. 访问 [http://localhost:8080](http://localhost:8080), 使用用户名 `root` 密码`exmaple` 登陆
